# 🎮 COLOR HUNTER

A moderrn and interactive color hunter game built with **HTML**, **CSS** and **JAVASCRIPT**.
Play and challenge your self

## ✨ Features

- Many levels
- Record highscore
- Record score
- Sound effect
- Restart game
- Smooth UI

## 🛠 Technologies

- HTML
- CSS
- JAVASCRIPT (VANILLA)

## 🚀 Getting started
1. Clone the repository
```bash
git clone https://github.com/fady-albert/color_hunter.git
```
2. Navigate to the project folder
```bash
cd color_hunter
```
3. Open `index.html` in your browser
